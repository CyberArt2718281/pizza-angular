Директивы - это некая сущность в angular, которая является классом добавляющим какое-то поведение для элементов 

@Directive({
	selector:'[test]'
})

Типы директив 
любой компонент это директива отличе в том что компонент использует декоратор Component а не Directive и за счет этого мы можем использовать в компонентах шаблоны тоесть по факту компонент это директива с шаблоном 
а у самих директив нет шаблона
1)Компоненты - директива с шаблоном компонент расширяет директиву 
2)Атрибутные используются в качестве атрибута для элемента изменяют поведение элементов к которым они применяются 

3)Структурные директивы - изменяют структуру DOM

Типы директив 
1) Встроенные - присутствует в Angular из коробки (NgIfm NgModel, NgFor, NgSwitch)
2)Кастомные директивы - созданные самостоятельно

Атрибутная директива - задается с помощью атрибута для элемента/компонента 
<component myDirective></>

Класс Renderer2 
Класс Angular для работы с DOM

Декоратор @HostListener
Позволяет задать функцию-обработчик для события элемента

Декоратор @HostListener - позволяет задать функцию-обработчик для события элемента

Передача значений в директиву 
<div class='main-action' myDirective [str]='text'></div>
@Directive({
	selector: '[myDirective]'
})
export class MyDirective{
	@input() str: string='';

}

Декоратор @HostBinding 
Позволяет привязать свойство класса к определенному свойству DOM-элемента
private _color = "";
@HostBinding('style.color')
get getColor(){
	return this._color;
}

Структурные Директивы
Они изменяют структуру DOM за счет добавления или удаления элементов html

2 аспекта:
Директиву можно использовать повторно и применять к любому элементу 
Вынести часть логики из компонента чтобы разгрузить его
